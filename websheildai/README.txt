Здравствуйте я Щедрин Ярослав, я учусь в Лицее №52. Я создал обработчик запросов на основе ИИ. Он фильтрует запросы на потенциально опасные и безопасные предпринимая действия по блокировке.

Целью моей работы было разработать своего рода "Фаервал" который сможет обучаться на получаемых запросах. Задачи были следующие: Первая - создание самого ИИ для обработки; Вторая - это сделать веб часть для обработки приходящих запросов, записывания их в базу данных и удобной их визуализации; Третья создание самой непосредственно базы данных для хранения запросов.

При запуске приложения создаётся локальный сервер(Приложение можно развернуть на любом сервере) который и занимается обработкой запросов. Каждый запрос отображается по адресу /api/v1/dashboard и весь список инцидентов по адресу /api/v1/dashboard/incidents

Для теста и демонстрации нужно запустить Python файл main.py в папке webshield-ai/app. После в папке Traffic_simulator/dist запустить test_traffic_simulator.exe.(Также можно запустить файл test_traffic_simulator.py в папке webshield-ai)
После перехода на https://127.0.0.1:8000 будет главная страница приложения. А последствия запуска программ можно просмотреть по адресам выше или кнопкам в пользовательском интерфейсе.
